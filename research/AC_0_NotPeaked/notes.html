Using CRT

We have from the CRT (Chinese Remainder Theorem), given any f(x)f(x)f(x) where 0≤f(x)≤L0 \leq f(x) \leq L0≤f(x)≤L,

a1=f(x) mod p1a_1 = f(x) \mod p_1a1​=f(x) mod p1​ a2=f(x) mod p2a_2 = f(x) \mod p_2a2​=f(x) mod p2​     ⋮\;\;\vdots⋮ ak=f(x) mod pka_k = f(x) \mod p_kak​=f(x) mod pk​

where p1⋅p2⋯⋅pk≥Lp_1 \cdot p_2 \dots \cdot p_k \geq Lp1​⋅p2​⋯⋅pk​≥L, then we can uniquely recover f(x)f(x)f(x). Note that k∈O(log⁡L)k \in O(\log L)k∈O(logL).

Moreover, we can find fixed constants {Mi},{Ni}\{M_i\}, \{N_i\}{Mi​},{Ni​} (from Wikipedia) such that

f(x)=∑i∈[k]aiMiNi.f(x) = \sum_{i \in [k]} a_i M_i N_i.f(x)=i∈[k]∑​ai​Mi​Ni​.

Now, this is cool beans.

Note that if we have the expectations for aia_iai​ we can get the expectation for fff!

CRT and Unique Identification

We can note that CRT guarantees us that there is a unique b1,…,bkb_1, \dots, b_kb1​,…,bk​ such that

∑i∈[k]biMiNi=1.\sum_{i \in [k]} b_i M_i N_i = 1.i∈[k]∑​bi​Mi​Ni​=1.

Lower Bounding the Expectation

So then, if we want to lower bound E[C(x)]\E[C(x)]E[C(x)] where x∼Dx \sim \distribx∼D and CCC is an AC0\ACZAC0 circuit, we simply need to find E[f′(x)]\E[f'(x)]E[f′(x)] where f′f'f′ is the polynomial that sends all outputs not equal to 1 to 0. We can see this as

E[f′(x)]=Pr⁡[f′(x)=1]≤Pr⁡[C(x)=1]=E[C(x)].\E[f'(x)] = \Pr[f'(x) = 1] \leq \Pr[C(x) = 1] = \E[C(x)].E[f′(x)]=Pr[f′(x)=1]≤Pr[C(x)=1]=E[C(x)].

Cool! So we can maybe find a lower bound. We can also use the same technique to upper bound the expectation by looking at the negation of CCC.
